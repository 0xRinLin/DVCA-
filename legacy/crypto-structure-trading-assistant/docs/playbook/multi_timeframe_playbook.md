# 多周期观察手册

本文根据 `docs/project_plan_v1.md` 创建，用于定义人工多周期观察流程。本文件不包含 Pine 代码，也不代表当前版本已经实现多周期自动计算。

## 目的

多周期观察用于回答三个问题：

1. 大方向是否支持当前交易方向。
2. 当前周期信号是否处在合理位置。
3. 小周期执行是否过早、过晚或追价。

## 周期分工

- 1D：判断大方向和重要风险背景。
- 4h：判断主结构、关键区域和趋势阶段。
- 1h：作为主要决策周期。
- 15m：用于观察细节入场、回踩、反抽和失效确认。

## 多头观察流程

1. 先看 1D / 4h 是否处于 UP_TREND、TRANSITION 或 RANGE。
2. 如果 4h 为 DOWN_TREND，不把 1h 普通反弹直接当成高置信 LONG。
3. 1h 需要出现 bullish_structure、有效 L-ZONE、PB-L 或 BO-L。
4. 15m 只用于优化位置，不用于推翻 1h / 4h 的结构。
5. 入场前必须确认结构失效位。

## 空头观察流程

1. 先看 1D / 4h 是否处于 DOWN_TREND、TRANSITION 或 RANGE。
2. 如果 4h 为 UP_TREND，不把 1h 普通回落直接当成高置信 SHORT。
3. 1h 需要出现 bearish_structure、有效 S-ZONE、PB-S 或 BO-S。
4. 15m 只用于优化位置，不用于推翻 1h / 4h 的结构。
5. 入场前必须确认结构失效位。

## 冲突处理

- 高周期趋势清晰，低周期反向信号只作为 warning 或回调观察。
- 高周期 RANGE，低周期信号默认降低置信度。
- 高周期 TRANSITION，低周期只允许轻量观察，不做高置信主信号。
- 高周期出现 EXHAUSTION，低周期信号必须重点检查 XL / XS 风险。

## 禁止事项

- 不用低周期单根 K 线否定高周期结构。
- 不在高周期结构不清晰时重仓依赖低周期信号。
- 不把 EXH-L 当成 SHORT。
- 不把 EXH-S 当成 LONG。
- 不把 XL / XS 当成反手入场。
