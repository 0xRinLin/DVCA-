# 学习资料摘要

本文根据 `docs/project_plan_v1.md` 完善，用于把学习资料转化为后续规格、状态机、信号规则和测试计划的共同依据。本文件只描述规则边界，不包含 Pine 代码。

## 资料来源范围

本项目吸收的技术分析资料范围包括：

1. 技术分析核心原则
2. 趋势判断
3. 道氏理论
4. 趋势线与趋势通道
5. 均线系统
6. Bollinger Band
7. MACD
8. 单根 K 线
9. K 线组合
10. 反转形态
11. 中继形态
12. 突破确认
13. 止损止盈
14. 盘感训练
15. 实战检查清单

这些资料在项目中不被直接当成买卖命令，而是被拆分成结构、过滤、区域、突破、信号和风险边界。

## 核心方法论

技术分析的目标不是预测绝对结果，而是在图表中识别更高概率的方向、位置和交易条件。

因此，本项目采用以下转换原则：

1. 先识别市场状态，再讨论信号。
2. 先确认道氏结构，再判断方向偏好。
3. 先区分趋势、震荡和过渡，再决定是否允许主信号。
4. 先定义失效条件，再输出入场参考。
5. 单一指标不得单独触发 LONG 或 SHORT。
6. warning 只能提示风险，不能直接反向开仓。
7. 主观经验只能辅助观察，不能替代客观规则。

## 1. 可以直接量化的内容

以下内容适合后续转化为确定性模块：

- confirmed Swing High / Swing Low
- HH / HL / LH / LL
- 最近 confirmed high / confirmed low
- lastConfirmedHigh
- lastConfirmedLow
- lastBullishInvalidation
- lastBearishInvalidation
- EMA 5 / 10 / 20 / 40 / 60
- 价格与 EMA20 的相对位置
- EMA10 与 EMA20 的排列关系
- Bollinger Band 上轨 / 中轨 / 下轨
- Bollinger Band 宽度变化
- MACD 方向与动能扩张 / 收缩
- ATR 14
- ATR buffer
- 收盘突破关键位
- 2% 突破参考规则
- K 线实体大小
- 突破后是否快速回到原区间
- 信号冷却周期
- label / line / box 数量控制

直接量化内容可以成为后续 Pine Script 的基础计算层，但仍需经过状态机过滤。

## 2. 可以半量化的内容

以下内容适合转化为评分、过滤器或 warning：

- K 线组合
- 中继形态
- 反转雏形
- 趋势衰竭
- 假突破风险
- 突破质量
- 回踩后的重新转强
- 反抽后的重新转弱
- 动能背离倾向
- 强震荡过滤
- 价格接近阻力或支撑后的风险提示

半量化内容不应单独触发主交易信号，必须与 Market Regime、Dow Structure 和 Breakout Score 结合。

## 3. 不适合直接自动化的内容

以下内容主观性较强，不适合直接做成 LONG / SHORT 或 XL / XS：

- 盘感
- 主观画线
- 复杂头肩形
- 复杂 W / M 形态
- 圆弧形
- 完整楔形判断
- 对新闻、情绪、突发事件的主观判断
- 对高低点重要性的人工选择

这些内容可以作为人工复盘提示，但不得成为自动信号的核心依据。

## 代码化分类表

| 资料类型 | 项目用途 | 后续表达方式 | 使用限制 |
| --- | --- | --- | --- |
| 道氏高低点 | 判断结构方向 | Dow Structure | 必须使用 confirmed pivot |
| 均线排列 | 趋势过滤 | Trend Filter | 不单独触发信号 |
| Bollinger Band | 趋势与波动辅助 | Volatility / Zone | 不单独触发信号 |
| MACD | 动能过滤 | Momentum | 不把金叉死叉当买卖命令 |
| ATR | 噪声过滤 | ATR buffer | 用于突破和区域容错 |
| K 线实体 | 突破质量 | Breakout Score | 只作为评分因素 |
| 衰竭形态 | 风险提示 | EXH-L / EXH-S | 只作为 warning |
| 盘感 | 人工辅助 | 文档提示 | 不自动化 |
