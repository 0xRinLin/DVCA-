# DVCA Gold Cases

本目录用于保存最标准、最教科书式的 DVCA 案例。

Gold Cases 不是原始截图仓库，也不是所有候选案例的备份。这里应该只放最有代表性的案例，用于未来 Signal Atlas、策略说明和指标升级复盘。

## 目录结构

```text
gold_cases/
├── BTC/
│   ├── LONG/
│   ├── PB/
│   ├── TC/
│   └── EXH/
├── ETH/
└── SOL/
```

## 收录标准

一个案例进入 Gold，必须至少满足一项：

- 信号链完整且标准。
- 截图清晰，前后 K 线足够。
- 能代表一个信号类型。
- 能解释一个重要失败模式。
- 能作为 Signal Atlas 的引用案例。

## 不收录的内容

- 只有 before，没有 after。
- 结果还不明确。
- 只是有趣但不能说明问题。
- 重复度高的普通截图。

## 推荐命名

```text
CASE-0007_BTC_30m_TC-L_success.md
CASE-0012_SOL_15m_SHORT_PB-S_fail.md
```
